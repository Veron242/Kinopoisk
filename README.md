# Kinopoisk
▶️Проект:


Кинопоиск –  один из самых популярных киносервисов в России. Предлагает бесплатный доступ к фильмам, телесериалам, сериалам и мультфильмам, с возможностью озвучки на русском языке. Кроме того, сайт имеет удобный интерфейс, возможность поиска фильмов по жанрам, актерам и режиссерам, рейтингу, году выпуска и другим параметрам. 

▶️Цель тестирования:

Проверить функциональность, удобство использования и производительность сайта Кинопоиск.

▶️Область тестирования:

Тестирование основных модулей сайта (поиск, регистрация, написание рецензии, рейтинг).  

▶️Основные функции:

Для проведения тестирования были выбраны следующие функции:

**1) Регистрация и авторизация**: Пользователи могут зарегистрироваться на сайте через email/телефон или зайти с уже имеющимся аккаунтом.

**2) Поиск фильмов по названию:** Пользователи могут искать фильмы по названию, жанру, году выпуска и актерам с возможностью фильтрации результатов.

**3) Написание отзыва:** После просмотра фильма пользователи могут оставить отзыв (рецензию).


Была составлена тестовая документация в формате тест-кейсов и чек-листов для проведения:

Смоук-тестирования
Функционального тестирования
Нефункционального тестирования(тестирование производительности, тестирование UI/UX и кросс-браузерное тестирование)
Тестирования API


▶️Какие инструменты были применены:

Test IT, Sitechko для подготовки тест-кейсов и чек-листов, Postman для создания коллекции API запросов, JMeter для тестирования производительности, Yonote для фиксации багов и ведения тест-плана и отчета.
